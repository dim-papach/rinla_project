RemoteAstrophotography.com Sample Observation of Messier 31
===========================================================

This ZIP file contains FITS observations of the Andromeda
Galaxy, or Messier 31, taken over 14 days. All of the files
have been aligned, so stacking should be simple.

These images were taken using the MicroObservatory and you
are free to use these images as you like.

https://RemoteAstrophotography.com
